# Focus & Insightful Timing (FIT) - Task Manager ⏱️

**Focus & Insightful Timing** is a clean, multi-page web application designed to help users boost productivity, manage time effectively, and reduce mental clutter. Built entirely with Vanilla JavaScript, HTML5, and CSS3.

## 🌟 Key Features
* **Full CRUD Functionality:** Users can seamlessly add, cross off (update), and delete daily tasks.
* **Data Persistence:** Integrated `localStorage` ensures that user tasks are saved directly in the browser and are not lost when the page is refreshed.
* **Responsive Design:** Custom CSS media queries guarantee a smooth user experience across desktop, tablet, and mobile devices.
* **Interactive UI:** Features hover-state action buttons (check/trash icons only appear when hovering over a task) and an accordion-style toggle feature on the "Benefits" page.
* **Safety Mechanisms:** Includes a browser confirmation prompt before deleting tasks to prevent accidental data loss.
* **Live Clock:** A real-time updating footer displaying the current date and time using JavaScript's `setInterval` and `Date` object.

## 🛠️ Technologies Used
* **HTML5:** Semantic structuring across three interconnected pages (Home, Benefits, To-Do List).
* **CSS3:** Custom styling, flexbox layouts, hover transitions, and mobile-first media queries.
* **Vanilla JavaScript (ES6+):** DOM manipulation, event listeners, array methods (`push`, `splice`, `forEach`), and JSON parsing/stringifying for local storage.

## 🧠 Developer Highlights
This project demonstrates proficiency in core front-end concepts without relying on external frameworks:
* **State Management:** Maintaining a central `myItemList` array and keeping the UI perfectly synced with the underlying data.
* **Event Handling:** Capturing both mouse clicks and keyboard events (e.g., pressing "Enter" to submit a task).
* **Dynamic Rendering:** Using JavaScript template literals to inject HTML dynamically based on the current state of the task array.

## 🚀 How to Run Locally
1. Clone this repository to your local machine.
2. Navigate to the project folder.
3. Simply open `landingpage.html` in your preferred web browser to start using the app (no local server required).
